How to use

  • Home 화면에서는 가로모드를 추천 합니다.
  • 타로 상담 진행을 3장 또는 5장으로 할지 선택해 주세요.
  • 기본 설정은 Major 카드만 사용합니다. Minor 카드를 사용하고 싶으면 선택해 주세요.
  • 상담 진행 전에는 꼭 [SHUFFLE] 버튼을 눌러주세요. 이후에 카드를 누르면 뽑힌 카드가 나타납니다.
  • 어떤 설정이든 설정 변경 후에는 꼭 [SHUFFLE] 버튼을 눌러주세요.
  • [OPEN ALL] 버튼을 누르면 모든 카드가 열립니다.

About this web page

  • 타로를 공부하거나 타로 상담을 진행하는데 있어 실제 카드가 없는 분들을 위해 제작 되었습니다.
